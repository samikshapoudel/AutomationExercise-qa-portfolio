# API Testing - Automation Exercise

## Status: In Progress

## Available APIs to Test
- GET /api/productsList — Get all products
- GET /api/brandsList — Get all brands
- POST /api/searchProduct — Search for a product
- POST /api/verifyLogin — Verify login with valid credentials
- POST /api/verifyLogin — Verify login with invalid credentials
- POST /api/createAccount — Create a new user account
- DELETE /api/deleteAccount — Delete a user account
- PUT /api/updateAccount — Update user account details
- GET /api/getUserDetailByEmail — Get user details by email

- Chrome DevTools
